# Codeforces

This folder contains my solutions to Codeforces problems.

Problems are organized based on their **difficulty rating** to track progress and improve step-by-step.

---

## 📂 Structure

- 800 → Easy / beginner-friendly problems
- 1000 → Slightly harder problems
- 1200+ → Increasing difficulty levels

---

## 🎯 Goal

- Build strong problem-solving skills
- Improve speed and accuracy during contests
- Master DSA concepts through practice
