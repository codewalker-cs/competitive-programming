# 🔍 Binary Search Problems

This folder contains problems that involve the **Binary Search** technique.

---

## 💡 What is Binary Search?

Binary Search is an efficient algorithm used to find an element or answer in a **sorted space** by repeatedly dividing the search range.

---

## ⚡ Key Concepts

- Search space reduction
- Monotonic functions
- Lower bound / Upper bound
- Answer-based binary search

---

## 🎯 Goal

- Master binary search patterns
- Learn to identify when binary search can be applied
- Improve optimization skills
